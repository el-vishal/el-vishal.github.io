Alpine Linux + XFCE + Remmina

Jun 16, 2023

Why?

I wanted to remote desktop my home machine through a low spec VM (via zerotier). The best way I was able to achieve it was using Alpine Linux with XFCE and Remmina

Steps

Step 1: Download Alpine

Whatever way to you are happy with - https://www.alpinelinux.org/

Step 2: Install XFCE

https://wiki.alpinelinux.org/wiki/Xfce

Step 3: Install Remmina (and FreeRDP - dependencies)

    apk add remmina    
    apk add freerdp

Step 4: Enjoy
